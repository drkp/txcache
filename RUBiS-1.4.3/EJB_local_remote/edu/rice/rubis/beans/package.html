This package contains all the Java Beans of the Rice University Bidding System (RUBiS). This is the session façade version using EJB 2.0 Local and remote interfaces. Current available entity beans are: Category, Region, User, Item, Bid, and Comment. There is one stateless session been named Query that is used to perform more complex queries on the database.
