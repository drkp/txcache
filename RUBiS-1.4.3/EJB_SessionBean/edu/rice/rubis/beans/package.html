This package contains all the Java Beans of the Rice University Bidding System (RUBiS). Current available session beans are: SB_BrowseCategories, SB_BrowseRegions, SB_SearchItemsByCategory, SB_SearchItemsByRegion, SB_ViewItem, SB_registerItem, SB_ViewUserInfo, SB_RegisterUser, SB_PutComment, SB_StoreComment, SB_BuyNow, SB_StoreBuyNow, SB_PutBid, SB_StoreBid, SB_ViewBidHistory, SB_AboutMe and SB_Auth.
